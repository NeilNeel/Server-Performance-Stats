# Server Performance Stats Script

## Overview
`server-stats.sh` is a simple Bash script that displays essential performance metrics for a Linux server.  
Itâ€™s designed for beginners to understand system monitoring commands and basic shell scripting.

## Features
- Total CPU usage  
- Memory usage (free vs used)  
- Disk usage (free vs used)  
- Top 5 processes by CPU usage  
- Top 5 processes by memory usage  
- (Optional) OS version, uptime, and load average
